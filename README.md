# 中文文字之情感分析
## 摘要
相比以往物資缺乏的年代，現代人雖較不愁吃穿，但經常會因為各式各樣的壓力而影響心理健康，進而造成各種心理疾病。  
本專題的目的在於建立一個可分析中文文字訊息情感的模型，並在未來將其應用於分析使用者的情緒變化，進而預測使用者的心理健康狀態。
## 研究方法
本研究製作了三種中文情緒分析模型與一個中文資料集，並進行初步的多元情緒分析：
### 中文情感辭典
1. 使用DepecheMood++ (DM++) 與中英辭典映射建立中文的情感標註辭典。
2. 使用CKIP工具對語句進行斷詞及實體辨識等處理。
3. 對照辭典與語句中的單詞，統計分數，進行情感標註。
### 基於最大熵的中文情感分析模型
1. 使用CKIP工具對資料集進行斷詞處理。
2. 產生特徵元組(標籤, 字詞)並統計其數量，建立經驗分布模型。
3. 產生一平均分布模型，開始模型訓練。
4. 比較當前模型與經驗分布模型之特徵期望值，調整各特徵之權重。
5. 若符合收斂條件，則訓練結束。
6. 對每個測試樣本，根據其字詞權重計算條件機率，以判斷為正向或負向樣本。
### 基於BiLSTM的中文情感分析模型
1. 使用CKIP工具對資料集進行斷詞處理。
2. 將訓練樣本之中文字詞轉換為詞向量並統一長度。
3. 建立一具備輸入層、嵌入層、反向LSTM層、正向LSTM層和輸出層之基本BiLSTM模型。
4. 進行模型訓練。
5. 使用測試資料集以及Accuracy評估模型表現。
### 新聞標題資料集製作
1. 利用網路爬蟲從數個新聞網擷取新聞標題。
2. 刪除無效或完全重複之資料。
3. 對資料集進行人工標註，將所有資料分為七個情緒類別。
### 心理諮商問答語料庫修改
1. 擷取由史丹佛大學、加利福尼亞大學洛杉磯分校、輔仁大學、Chatopera及志願者製作的心理諮商問答語料庫。
2. 僅保留語料庫中的主要文字訊息與心理疾病分類標註。
3. 將資料翻譯為繁體中文，並修改其用詞為臺灣用法。
### 最大熵中文情感分析模型測試
1. 對每個情緒類別各自建立一個最大熵分類模型。
2. 使用新聞標題資料集訓練模型。
3. 使用模型對修改後的心理諮商問答語料庫進行情感分析。

## 結論
根據DM++製作而成的中文情感辭典僅是依照辭典進行字詞的對應，而如此簡易的方法其準確度仍有六成，雖然稱不上是高準確度，但以此方法而言，是超乎預期的好結果，可見此種方式能夠快速建立一個有相當準確度的辭典。若能完全從中文資料建立辭典，應該就能避免文化和字典所造成的差異，讓準確度再提升。

基於最大熵的情感分類模型僅使用了數千筆資料就能得到有八成以上準確度的模型，相當適用於資料集不足的狀況。但若要標註複數個情緒，則須為每個情緒訓練一個模型才能保有其準確度。且若以字詞作為特徵單位，則無法考慮到不同字詞相鄰時語意改變的情況，使得最大熵模型在實務應用上仍有所不足，這點在情緒向量標註時就能觀測到。

基於BiLSTM的情感分類模型擁有相當優秀的能力，在資料量如此不足的情況下仍能達到七成的準確度，但也因此表現不如最大熵模型。

而目前在情感分析領域，中文資料相當缺乏，導致模型訓練的困難。縱使本研究有自行製作新聞標題資料集並進行情感標註，但此資料集因只有一人進行標註，有失公平性，無法保證其測試結果。
